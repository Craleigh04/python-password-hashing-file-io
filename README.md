# Python Password Hashing and File I/O

This project demonstrates Python fundamentals through password hashing, file handling, and dictionary processing using modular functions and exception handling.

## Overview
The program performs three key operations:
1. Hashes a user-provided password using the hashlib module
2. Writes system information from a dictionary to a file
3. Reads the file and reconstructs the dictionary

## Features
- Password hashing using MD5
- File writing and reading
- Dictionary processing
- Modular function design
- Exception handling with try/except

## Skills Demonstrated
- Python programming
- File I/O operations
- Hashing and basic security concepts
- Dictionaries and data structures
- Functions and modular code design
- Error handling

- `main.py`
- `security_functions.py`
